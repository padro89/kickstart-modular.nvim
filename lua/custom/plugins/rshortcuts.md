| Action                       | Default Keybinding (assuming `vim.g.R_leader = ','`) |
| ---------------------------- | ---------------------------------------------------- |
| Start R (REPL console)       | `,rf` *(Run File)*                                   |
| Send current line            | `,l`                                                 |
| Send selected lines (Visual) | `,l` (while in Visual Mode)                          |
| Send current chunk           | `,cc`                                                |
| Send the entire file         | `,rf`                                                |
| Send paragraph (block)       | `,p`                                                 |
| Start R Markdown support     | `,rm` *(in Rmd)*                                     |
| Close R session              | `,q`                                                 |
